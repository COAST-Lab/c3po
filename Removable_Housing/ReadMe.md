Objectives:
- Create an upgraded housing that allows easy access to electrical components and sensors
  while maintaining water tight reliability
- Add opening for pressure transducer
  

## Design Improvements
 **Removable End Caps**  
  The updated design features two removable caps:
  - Sensor-facing cap
  - Electronics/battery access cap  
  This allows for easy access to internal components without damaging the housing.

 **Exterior Fastening**
- Each cap was designed with a circular overhang with 2 (oppisite side) 1/4 inch holes
- Each hole is used for a 1/4 inch screw to lock-in endcaps
- Screws are fastened on oppisite sides with hex nuts

 **O-Ring Sealing System**  
- The housing uses O-ring grooves to maintain a watertight seal while still allowing repeated
  opening and closing.

  **Pressure Sensor Integration**  
- An additional port has been incorporated to accommodate a pressure sensor, enabling depth
  measurements to be included in the system’s output data.

  
## Benefits
- Non-destructive access to electronics and sensors  
- Improved maintenance and upgrade capability  
- Reusable housing design  
- Expanded data collection through depth sensing

## Notes
- This design is intended for iterative testing. Fit, sealing performance, and materials 
(sanding tolerances and O-ring compression) should be validated during assembly and field use.
